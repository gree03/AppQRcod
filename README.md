# AppQRcod

Telegram-бот для отображения и редактирования расписания студентов 3 курса направления 35.03.06 «Электрооборудование и электротехнологии».

## Возможности

- Получение актуального номера учебной недели с сайта БГАУ и отображение расписания на сегодня (`/сегодня`) и завтра (`/завтра`).
- Автоматическая рассылка расписания каждому пользователю, запустившему бота (по умолчанию ежедневно в 07:00 по часовому поясу `Asia/Yekaterinburg`).
- Управление расписанием через диалог `/edit`: выбор недели, дня, занятия и редактирование его полей (время, предмет, преподаватель, аудитория, подгруппа, недели).
- Хранение расписания и списка пользователей в JSON-файлах (`data/schedule.json`, `data/users.json`).

## Запуск бота

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Установите переменную окружения `BOT_TOKEN` со значением токена вашего Telegram-бота. При необходимости можно задать `BOT_TIMEZONE` (по умолчанию используется `Asia/Yekaterinburg`).

3. Запустите бота:

   ```bash
   python main.py
   ```

При старте бот предложит клавиатуру с кнопками `/сегодня`, `/завтра` и «Изменить расписание». Команда `/cancel` отменяет режим редактирования.
