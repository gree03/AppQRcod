# AppQRcod

Минимальный пример приложения на Flask для заказа еды через QR-код. Запуск:

```
pip install -r requirements.txt
python app.py
```

Приложение включает выбор режима, просмотр меню, корзину и оформление заказа.

Для привязки к столу используйте ссылки вида `/table/<номер>` (1–10), что удобно для QR-кодов.
