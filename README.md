# AppQRcod

Telegram-бот для отображения и редактирования расписания студентов 3 курса направления 35.03.06 «Электрооборудование и электротехнологии».

## Возможности

- Получение номера текущей учебной недели с сайта БГАУ (с запасным подсчётом от первого понедельника сентября) и отображение расписания на сегодня (`/сегодня`) и завтра (`/завтра`).
- Просмотр расписания на любую учебную неделю через команду `/неделя` и интерактивный выбор недели.
- Личные напоминания о расписании: утреннее (07:00 — на сегодня) и вечернее (20:00 — на завтра) каждый студент включает в своём чате с ботом; в беседах напоминания отправляются по умолчанию.
- Управление расписанием через диалог `/edit`: выбор недели, дня, занятия и редактирование его полей (время, предмет, преподаватель, аудитория, подгруппа, недели). Изменять расписание могут только @Greegan03, @SergoBranches и @EKiritoo.
- Хранение расписания и списка пользователей в JSON-файлах (`data/schedule.json`, `data/users.json`).

## Запуск бота

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Создайте в корне проекта файл `config.txt` (можно взять за основу `config.txt.example`) и укажите в нём токен бота. Допускаются форматы `BOT_TOKEN=123456:ABC` или просто сам токен в первой непустой строке.

3. При необходимости задайте переменную окружения `BOT_TIMEZONE` (по умолчанию используется `Asia/Yekaterinburg`).

4. Запустите бота:

   ```bash
   python main.py
   ```

При старте бот предложит клавиатуру с кнопками `/сегодня`, `/завтра`, «Расписание на неделю», «Изменить расписание» и «Напоминания». Команды `/неделя` и `/notify` дублируют соответствующие кнопки. Команда `/cancel` отменяет режим редактирования.
